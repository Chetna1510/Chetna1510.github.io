QuickCart Update – WooCommerce Cart Auto-Update Plugin

Technology Stack: PHP, WordPress, JavaScript (jQuery), API, AJAX

Description:

This WooCommerce plugin automatically updates cart totals when quantity changes on the cart page, eliminating the need to manually click the “Update Cart” button.

Features:

  • Automatically detects quantity changes in the WooCommerce cart.
  • Uses AJAX to update cart contents without full page reload.
  • Added a timed delay to prevent unnecessary multiple updates on rapid quantity changes.
  • Loads JavaScript only on the cart page for performance optimization.
